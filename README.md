# MacGraph
The MacGraph network. An Irish attempt at intelligence. Puns not included.

Almost nothing is written yet.


## Running the code
### Prerequisites

We use the pipenv dependency/virtualenv framework:
```shell
$ pipenv install
$ pipenv shell
(mac-graph-sjOzWQ6Y) $
```

### Testing

Test the read cell

```python
python -m mac-graph.cell.test_read_cell
```
